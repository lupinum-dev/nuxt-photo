# PhotoAlbum

The `PhotoAlbum` component renders a collection of photos in a grid using one of four layout algorithms — rows, columns, masonry, or bento. It includes an integrated lightbox by default.

::photo-playground
::

## Usage

### Basic

```vue
<script setup>
const photos = [
  { id: 1, src: '/photos/a.jpg', width: 1600, height: 900 },
  { id: 2, src: '/photos/b.jpg', width: 900, height: 1200 },
  { id: 3, src: '/photos/c.jpg', width: 1000, height: 1000 },
]
</script>

<template>
  <PhotoAlbum :photos="photos" />
</template>
```

### With layout options

::code-group
```vue [String shorthand]
<PhotoAlbum :photos="photos" layout="masonry" />
```

```vue [Object form]
<PhotoAlbum
  :photos="photos"
  :layout="{ type: 'rows', targetRowHeight: 250 }"
  :spacing="12"
/>
```

```vue [Responsive]
<PhotoAlbum
  :photos="photos"
  :layout="{ type: 'columns', columns: responsive({ 0: 2, 768: 3, 1200: 4 }) }"
  :spacing="responsive({ 0: 4, 768: 8, 1200: 12 })"
/>
```
::

### Inside a PhotoGroup

When placed inside a [`PhotoGroup`](/components/photo-group), the album's photos join the group's shared lightbox:

```vue
<PhotoGroup>
  <PhotoAlbum :photos="landscapes" layout="rows" />
  <PhotoAlbum :photos="portraits" layout="columns" />
</PhotoGroup>
```

### With custom thumbnails

```vue
<PhotoAlbum :photos="photos">
  <template #thumbnail="{ photo, index, width, height, hidden }">
    <div :style="{ opacity: hidden ? 0 : 1 }">
      <img :src="photo.thumbSrc || photo.src" :alt="photo.alt" />
      <span class="badge">{{ index + 1 }}</span>
    </div>
  </template>
</PhotoAlbum>
```

### With CMS data

```vue
<script setup>
import type { PhotoAdapter } from '@nuxt-photo/core'

const cmsPhotos = await fetchFromCMS()

const adapter: PhotoAdapter = (item) => ({
  id: item.uuid,
  src: item.file.url,
  width: item.file.width,
  height: item.file.height,
  alt: item.title,
  caption: item.title,
  description: item.description,
})
</script>

<template>
  <PhotoAlbum :photos="cmsPhotos" :photo-adapter="adapter" />
</template>
```

## Props

::field-group
  ::field{name="photos" type="PhotoItem[] | any[]" required}
  Array of photos. When using `photoAdapter`, items can be any shape.
  ::

  ::field{name="photoAdapter" type="PhotoAdapter"}
  Transforms each item in `photos` into a `PhotoItem`. Use when feeding CMS or API data directly.
  ::

  ::field{name="layout" type='AlbumLayout | "rows" | "columns" | "masonry" | "bento"' defaultValue='"rows"'}
  Layout algorithm. Pass a string for defaults, or an object for full control. See [Layout Options](#layout-options) below.
  ::

  ::field{name="spacing" type="ResponsiveParameter<number>" defaultValue="8"}
  Gap between images in pixels. Accepts a static number or a `responsive()` function.
  ::

  ::field{name="padding" type="ResponsiveParameter<number>" defaultValue="0"}
  Outer padding around each image in pixels. Accepts a static number or a `responsive()` function.
  ::

  ::field{name="defaultContainerWidth" type="number"}
  Assumed container width for SSR. When set, the JS layout runs server-side so the rendered HTML matches hydration — eliminating CLS. Combine with `breakpoints`.
  ::

  ::field{name="breakpoints" type="readonly number[]"}
  Snaps the observed container width to the nearest breakpoint. Prevents re-layout on sub-pixel fluctuations and scrollbar changes.
  ::

  ::field{name="sizes" type='{ size: string; sizes?: Array<{ viewport: string; size: string }> }'}
  `<img sizes>` hint for responsive images in the rows layout. `size` describes the container width (e.g., `'100vw'`).
  ::

  ::field{name="adapter" type="ImageAdapter"}
  Custom image adapter for generating `src`, `srcset`, and `sizes` attributes.
  ::

  ::field{name="lightbox" type="boolean | Component" defaultValue="true"}
  Whether to enable the lightbox. Pass `false` to disable, `true` for default, or a custom component.
  ::

  ::field{name="transition" type="LightboxTransitionOption"}
  Transition mode for lightbox open/close. See [Transitions guide](/guides/transitions).
  ::

  ::field{name="itemClass" type="string"}
  Extra CSS classes for each album item wrapper `<div>`.
  ::

  ::field{name="imgClass" type="string"}
  Extra CSS classes for each album `<img>` element.
  ::
::

## Layout Options

The `layout` prop accepts a string shorthand or an object. Each layout type has its own options:

### Rows

```vue
<PhotoAlbum :photos="photos" :layout="{ type: 'rows', targetRowHeight: 280 }" />
```

::field-group
  ::field{name="targetRowHeight" type="ResponsiveParameter<number>" defaultValue="300"}
  Ideal row height in pixels. The algorithm adjusts photo widths to fill each row while keeping heights close to this value.
  ::
::

### Columns

```vue
<PhotoAlbum :photos="photos" :layout="{ type: 'columns', columns: 3 }" />
```

::field-group
  ::field{name="columns" type="ResponsiveParameter<number>" defaultValue="3"}
  Number of columns.
  ::
::

### Masonry

```vue
<PhotoAlbum :photos="photos" :layout="{ type: 'masonry', columns: 3 }" />
```

::field-group
  ::field{name="columns" type="ResponsiveParameter<number>" defaultValue="3"}
  Number of columns. Photos are placed in the shortest column.
  ::
::

### Bento

```vue
<PhotoAlbum :photos="photos" :layout="{ type: 'bento', columns: 3, rowHeight: 280 }" />
```

::field-group
  ::field{name="columns" type="ResponsiveParameter<number>" defaultValue="3"}
  Number of grid columns.
  ::

  ::field{name="rowHeight" type="ResponsiveParameter<number>" defaultValue="280"}
  Height of each grid row in pixels.
  ::

  ::field{name="sizing" type='"auto" | "pattern" | "manual"' defaultValue='"auto"'}
  How column/row spans are determined. `'auto'` uses heuristics based on aspect ratio.
  ::

  ::field{name="patternInterval" type="number" defaultValue="5"}
  When `sizing` is `'pattern'`, how many photos before the pattern repeats.
  ::
::

## Slots

| Slot | Props | Description |
|---|---|---|
| `thumbnail` | `{ photo, index, width, height, hidden }` | Custom thumbnail. Replaces the default `<img>`. `hidden` is `true` during FLIP transitions. |
| `toolbar` | `{ activeIndex, activePhoto, count }` | Custom lightbox toolbar. |
| `caption` | `{ photo, activeIndex }` | Custom lightbox caption. |
| `slide` | `{ photo, index, width, height }` | Custom lightbox slide content. |
