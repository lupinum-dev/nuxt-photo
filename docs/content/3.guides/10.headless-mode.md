# Headless Mode

PhotoGroup's scoped slot lets you build **any layout you want** — CSS Grid, Flexbox, a horizontal scroller, a map — while still getting the full lightbox experience.

::collection-playground{initial-tab="headless"}
::

## Basic Headless Usage

Pass `:photos` to PhotoGroup and use the scoped slot:

```vue
<PhotoGroup :photos="photos" v-slot="{ open, setThumbRef }">
  <div class="my-grid">
    <div
      v-for="(photo, i) in photos"
      :key="photo.id"
      :ref="setThumbRef(i)"
      class="my-grid-item"
      @click="open(i)"
    >
      <img :src="photo.thumbSrc || photo.src" :alt="photo.alt" />
    </div>
  </div>
</PhotoGroup>
```

## Slot Props

::field-group
  ::field{name="open" type="(photoOrIndex?: PhotoItem | number) => Promise<void>"}
  Opens the lightbox at the given index or photo.
  ::

  ::field{name="setThumbRef" type="(index: number) => (el: HTMLElement | null) => void"}
  Returns a ref callback for the thumbnail at the given index. Bind this to your thumbnail element to enable FLIP transitions.
  ::
::

## Why Use setThumbRef?

`setThumbRef` registers the thumbnail's DOM element with the lightbox so it can:

1. Capture the thumbnail's position for FLIP open transitions
2. Hide the thumbnail during the transition (prevent duplication)
3. Animate back to the thumbnail on close

Without `setThumbRef`, the lightbox still works but falls back to fade transitions.

::code-group
```vue [With FLIP transitions]
<div :ref="setThumbRef(i)" @click="open(i)">
  <img :src="photo.src" />
</div>
```

```vue [Without FLIP (fade only)]
<div @click="open(i)">
  <img :src="photo.src" />
</div>
```
::

## Programmatic Open via Ref

Access `open()` and `close()` imperatively via a template ref:

```vue
<script setup>
const groupRef = ref()

function openFromOutside(index: number) {
  groupRef.value?.open(index)
}
</script>

<template>
  <button @click="openFromOutside(0)">Open Gallery</button>

  <PhotoGroup ref="groupRef" :photos="photos" v-slot="{ open, setThumbRef }">
    <div v-for="(photo, i) in photos" :key="photo.id">
      <div :ref="setThumbRef(i)" @click="open(i)">
        <img :src="photo.src" :alt="photo.alt" />
      </div>
    </div>
  </PhotoGroup>
</template>
```

## Examples

### Horizontal scroll gallery

```vue
<PhotoGroup :photos="photos" v-slot="{ open, setThumbRef }">
  <div style="display: flex; overflow-x: auto; gap: 8px;">
    <div
      v-for="(photo, i) in photos"
      :key="photo.id"
      :ref="setThumbRef(i)"
      style="flex-shrink: 0; cursor: pointer;"
      @click="open(i)"
    >
      <img
        :src="photo.src"
        :alt="photo.alt"
        style="height: 200px; width: auto;"
      />
    </div>
  </div>
</PhotoGroup>
```

### Single featured photo with lightbox

```vue
<PhotoGroup :photos="photos" v-slot="{ open, setThumbRef }">
  <div :ref="setThumbRef(0)" @click="open(0)" style="cursor: pointer;">
    <img :src="photos[0].src" :alt="photos[0].alt" style="width: 100%;" />
    <p>Click to browse all {{ photos.length }} photos</p>
  </div>
</PhotoGroup>
```

### With Tailwind grid

```vue
<PhotoGroup :photos="photos" v-slot="{ open, setThumbRef }">
  <div class="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4">
    <div
      v-for="(photo, i) in photos"
      :key="photo.id"
      :ref="setThumbRef(i)"
      class="cursor-pointer rounded-lg overflow-hidden"
      @click="open(i)"
    >
      <img
        :src="photo.src"
        :alt="photo.alt"
        class="w-full h-48 object-cover"
      />
    </div>
  </div>
</PhotoGroup>
```
