# Shared Lightbox

By default, each `PhotoAlbum` creates its own lightbox. To navigate between photos across multiple albums or scattered `Photo` components, wrap them in a [`PhotoGroup`](/components/photo-group).

## Multiple Albums

```vue
<PhotoGroup>
  <h2>Landscapes</h2>
  <PhotoAlbum :photos="landscapes" layout="rows" />

  <h2>Portraits</h2>
  <PhotoAlbum :photos="portraits" layout="columns" />

  <h2>Abstract</h2>
  <PhotoAlbum :photos="abstract" layout="masonry" />
</PhotoGroup>
```

Clicking any photo opens a single lightbox that lets you navigate through **all** photos across all three albums.

## Scattered Photos

```vue
<PhotoGroup>
  <div class="hero">
    <Photo :photo="hero" />
  </div>

  <p>Some article text here...</p>

  <div class="inline-photos">
    <Photo :photo="detail1" />
    <Photo :photo="detail2" />
  </div>

  <p>More text...</p>

  <Photo :photo="conclusion" />
</PhotoGroup>
```

All `Photo` components auto-register with the group, regardless of their position in the DOM.

## Mixed Albums and Photos

```vue
<PhotoGroup>
  <Photo :photo="featuredPhoto" />
  <PhotoAlbum :photos="galleryPhotos" layout="rows" />
</PhotoGroup>
```

Both the standalone `Photo` and the `PhotoAlbum`'s photos share the same lightbox.

## Cross-Album Navigation

When navigating through photos in a shared lightbox, the transition system tracks which thumbnail to animate back to. If the active photo belongs to a different album than the one you started from, the close transition falls back to a fade (since the FLIP target is in a different scroll position or album).

## Customizing the Shared Lightbox

Pass slots on `PhotoGroup` to customize the lightbox:

```vue
<PhotoGroup>
  <template #toolbar="{ activeIndex, count }">
    <div class="my-toolbar">{{ activeIndex + 1 }} / {{ count }}</div>
  </template>

  <template #caption="{ photo }">
    <div class="my-caption">
      <h3>{{ photo.caption }}</h3>
      <p>{{ photo.description }}</p>
    </div>
  </template>

  <PhotoAlbum :photos="photos" />
</PhotoGroup>
```

## Opting Photos Out

Use `lightbox-ignore` on a `Photo` to exclude it from the group:

```vue
<PhotoGroup>
  <Photo :photo="photo1" />
  <Photo :photo="photo2" lightbox-ignore />  <!-- not in the lightbox -->
  <Photo :photo="photo3" />
</PhotoGroup>
```

Photo 2 renders as a plain image with no click interaction.
