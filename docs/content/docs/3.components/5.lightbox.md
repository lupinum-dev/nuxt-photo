---
title: Lightbox
description: Recipe lightbox component for customizing captions, actions, counters, and slides without rebuilding the runtime.
---

`<Lightbox>` is the recipe lightbox component used by Nuxt Photo's higher-level components. Use it when you want to keep the built-in structure and behavior, but change the visible UI.

This component is **not** a standalone gallery. It expects an active lightbox context from `<Photo>`, `<PhotoAlbum>`, `<PhotoGroup>`, `<PhotoCarousel>`, or a custom component built with `useLightboxProvider()`.

## Import

```ts
import { Lightbox } from '@nuxt-photo/recipes'
```

## When to use

Reach for `<Lightbox>` when you want to:

- change the caption layout
- replace the top-bar actions
- render a different counter
- customize slide markup while keeping the built-in lightbox flow

If you need different state wiring or a different primitive structure, move to [Build a custom lightbox](/docs/guides/custom-lightbox).

## Slots

| Slot      | Slot props                                                                     | Description                                    |
| --------- | ------------------------------------------------------------------------------ | ---------------------------------------------- |
| `counter` | `{ activeIndex, count }`                                                       | Replace the `3 / 12` counter.                  |
| `actions` | `{ prev, next, close, toggleZoom, isZoomedIn, zoomAllowed, controlsDisabled }` | Replace the top-bar buttons.                   |
| `caption` | `{ photo, index }`                                                             | Replace the caption area below the slide.      |
| `slide`   | `{ photo, index, width, height }`                                              | Replace the slide content for the active view. |

## Example

This is the smallest wrapper that changes the caption and action buttons, then reuses the rest of the recipe lightbox:

```vue [components/MyLightbox.vue]
<script setup lang="ts">
import { Lightbox } from '@nuxt-photo/recipes'
</script>

<template>
  <Lightbox>
    <template #caption="{ photo, index }">
      <div class="my-caption">
        <div class="text-xs opacity-60">Photo {{ index + 1 }}</div>
        <h2 v-if="photo?.caption">{{ photo.caption }}</h2>
        <p v-if="photo?.description">{{ photo.description }}</p>
      </div>
    </template>

    <template
      #actions="{
        prev,
        next,
        close,
        toggleZoom,
        isZoomedIn,
        zoomAllowed,
        controlsDisabled,
      }"
    >
      <button :disabled="controlsDisabled" @click="prev">Prev</button>
      <button :disabled="controlsDisabled" @click="next">Next</button>
      <button
        :disabled="controlsDisabled || !zoomAllowed"
        @click="toggleZoom()"
      >
        {{ isZoomedIn ? 'Fit' : 'Zoom' }}
      </button>
      <button :disabled="controlsDisabled" @click="close">Close</button>
    </template>
  </Lightbox>
</template>
```

Use that component through `:lightbox` on a recipe component:

```vue
<script setup lang="ts">
import MyLightbox from '~/components/MyLightbox.vue'
</script>

<template>
  <PhotoAlbum :photos="photos" :lightbox="MyLightbox" />
</template>
```
