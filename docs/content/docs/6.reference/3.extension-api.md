Use this page to look up the supported customization surfaces after the engine cutover.

## Surfaces

| Surface                | Import from          | Use it for                                                           | Notes                                            |
| ---------------------- | -------------------- | -------------------------------------------------------------------- | ------------------------------------------------ |
| `useLightboxProvider`  | `@nuxt-photo/vue`    | Building a custom lightbox UI with Vue primitives                    | Primary advanced Vue entrypoint                  |
| Lightbox primitives    | `@nuxt-photo/vue`    | Custom overlays, viewports, slides, controls, captions, and triggers | Stay on this layer for custom Vue UI             |
| `LightboxComponentKey` | `@nuxt-photo/vue`    | Replacing the default recipe lightbox globally                       | Provided through Vue injection                   |
| `ImageAdapterKey`      | `@nuxt-photo/vue`    | Providing a default image adapter                                    | Use when module-level image wiring is not enough |
| `LightboxDefaultsKey`  | `@nuxt-photo/vue`    | Setting shared default lightbox options                              | Vue-only customization hook                      |
| `createLightboxEngine` | `@nuxt-photo/engine` | Using the framework-free lightbox runtime directly                   | Lowest public layer                              |

## When to use which

- Build a custom lightbox UI with `useLightboxProvider` plus Vue primitives.
- Override the default recipe lightbox globally with `LightboxComponentKey`.
- Provide a default image adapter with `ImageAdapterKey`.
- Drop to `@nuxt-photo/engine` only when you need the orchestration layer below Vue.

## Examples

### Global lightbox override

```ts
import { LightboxComponentKey } from '@nuxt-photo/vue'
import MyLightbox from '~/components/MyLightbox.vue'

export default defineNuxtPlugin((nuxtApp) => {
  nuxtApp.vueApp.provide(LightboxComponentKey, MyLightbox)
})
```

### Engine runtime

```ts
import { createLightboxEngine } from '@nuxt-photo/engine'
import type { PhotoItem } from '@nuxt-photo/core'

const photos: PhotoItem[] = [
  { id: 'desert', src: '/photos/desert.jpg', width: 1280, height: 800 },
]

const engine = createLightboxEngine({ photos })
engine.open(0)
engine.markOpened()
```
