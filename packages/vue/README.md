# @nuxt-photo/vue

Vue primitives and composables for Nuxt Photo.

Stable root exports cover the intended public surface. `@nuxt-photo/vue/extend` is public but lower-stability and meant for advanced overrides.

## Install

```bash
pnpm add @nuxt-photo/vue @nuxt-photo/core
```

## Public surface

- primitives like `LightboxRoot`, `LightboxOverlay`, `LightboxViewport`, `PhotoTrigger`, and `PhotoImage`
- composables like `useLightbox`, `useLightboxProvider`, and `responsive`
- lower-stability extension exports from `@nuxt-photo/vue/extend`
