# @nuxt-photo/vue

Vue primitives and composables for Nuxt Photo.

Use the root `@nuxt-photo/vue` entrypoint for both normal usage and advanced customization.

## Install

```bash
pnpm add @nuxt-photo/vue @nuxt-photo/core @nuxt-photo/engine
```

## Public surface

- primitives like `LightboxRoot`, `LightboxOverlay`, `LightboxViewport`, `PhotoTrigger`, and `PhotoImage`
- composables like `useLightbox`, `useLightboxProvider`, and `responsive`
- stable injection keys like `LightboxComponentKey`, `ImageAdapterKey`, and `LightboxDefaultsKey`
