# @nuxt-photo/core

Framework-free layout, image, transition, and photo-model logic for Nuxt Photo.

Use this package when you want the underlying layout math and shared types without the Vue or Nuxt layers.

## Install

```bash
pnpm add @nuxt-photo/core
```

## Public surface

- `PhotoItem` and related types
- layout helpers like `responsive`
- framework-free image and geometry helpers that are intentionally exported from the root
